Contactly

Developed a phoneBook web application using Golang that allows users to synchronize their icloud contact with google contacts. Using Gorilla Mux for HTTP routing, MySQL database for storage, swagger for API documentation and html/css/javascript for the frontend. Technologies used - Golang, MySQL, Javascript, HTML, CSS, Json, Heroku, CircleCi
